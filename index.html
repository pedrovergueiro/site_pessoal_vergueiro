Quem sou eu

Estudo Inteligência Artificial na UNIFEI (Universidade Federal de Itajubá) e pesquiso compressão de redes neurais via Neural Architecture Search e Aprendizado Contínuo. O resultado principal até agora é o dNATY — um engine open-source que comprime qualquer modelo PyTorch com uma linha de código, sem precisar retreinar.

Também sou fundador da Vergueiro Tech, onde entrego sistemas web e de IA para clientes reais: um bot de WhatsApp para barbearia, uma loja online com fluxo automatizado de pedidos, dashboards de controle para indústria farmacêutica, e mais. Gosto de fechar o ciclo: pesquisa → código → produto.

Projeto Principal · Open-Source + SaaS

dNATY — Neural Network Compression

Engine de compressão neural via Neural Architecture Search + Continual Learning. Comprime qualquer modelo PyTorch com uma chamada de função, sem retreinar do zero. Foco em Edge ML: câmeras, drones, dispositivos CPU-only.

−46.5% FLOPs
1.6× Speedup
6.9× vs EWC
NAS evolutivo + operadores de mutação customizados
Continual Learning sem catastrophic forgetting
SaaS em produção — Railway, FastAPI, PostgreSQL, Redis
